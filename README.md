# Netflix Data Analysis

Exploratory data analysis and cleaning of the Netflix Movies and TV Shows dataset.

## Objective
Analyze the Netflix catalog to identify content trends, growth over time, and leading production countries.

## Tools
- Python
- Pandas
- Matplotlib

## Process
1. Data loading and exploration
2. Missing values handling and data cleaning
3. Feature engineering (year and month extraction)
4. Data visualization and conclusions

## Key Findings
- Movies represent over 60% of Netflix's total content catalog.
- Content additions grew rapidly from 2015 to 2019, peaking in 2019. The 2020 pandemic further accelerated the downward trend in content additions.
- The USA is the top content producer, followed by India and the United Kingdom. Notably absent is China, the world's most populous country.
